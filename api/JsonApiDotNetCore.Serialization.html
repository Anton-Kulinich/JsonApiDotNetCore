Namespace JsonApiDotNetCore.Serialization

Classes

AtomicOperationsResponseSerializer

Server serializer implementation of BaseSerializer for atomic:operations responses.

BaseDeserializer

Abstract base class for deserialization. Deserializes JSON content into Documents and constructs instances of the resource(s) in the document body.

BaseSerializer

Abstract base class for serialization. Uses IResourceObjectBuilder to convert resources into ResourceObjects and wraps them in a Document.

EmptyResponseMeta

Provides a method to obtain global JSON:API meta, which is added at top-level to a response Document. Use GetMeta(TResource) to specify nested metadata per individual resource.

FieldsToSerialize

Responsible for getting the set of fields that are to be included for a given type in the serialization result. Typically combines various sources of information, like application-wide and request-wide sparse fieldsets.

JsonApiReader

The deserializer of the body, used in ASP.NET Core internally to process FromBody.

JsonApiSerializationException

The error that is thrown when (de)serialization of a JSON:API body fails.

JsonApiWriter

Formats the response data used (see https://docs.microsoft.com/en-us/aspnet/core/web-api/advanced/formatting?view=aspnetcore-3.0). It was intended to have as little dependencies as possible in formatting layer for greater extensibility.

RequestDeserializer

Server deserializer implementation of the BaseDeserializer.

ResponseSerializer<TResource>

Server serializer implementation of BaseSerializer for resources of a specific type.

ResponseSerializerFactory

A factory class to abstract away the initialization of the serializer from the ASP.NET Core formatter pipeline.
