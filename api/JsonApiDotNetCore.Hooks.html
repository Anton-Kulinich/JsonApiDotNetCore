Transient service responsible for executing Resource Hooks as defined in ResourceDefinition<TResource>. see methods in IReadHookExecutor, IUpdateHookExecutor and IOnReturnHookExecutor for more information.

Uses JsonApiDotNetCore.Hooks.TraversalHelper for traversal of nested entity data structures. Uses JsonApiDotNetCore.Hooks.HookExecutorHelper for retrieving meta data about hooks, fetching database values and performing other recurring internal operations.
