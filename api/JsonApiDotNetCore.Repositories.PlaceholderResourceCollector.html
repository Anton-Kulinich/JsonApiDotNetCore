Class PlaceholderResourceCollector

Creates placeholder resource instances (with only their ID property set), which are added to the Entity Framework Core change tracker so they can be used in relationship updates without fetching the resource. On disposal, the created placeholders are detached, leaving the change tracker in a clean state for reuse.
