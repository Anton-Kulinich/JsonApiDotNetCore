Class QueryExpressionParser

The base class for parsing query string parameters, using the Recursive Descent algorithm.

public abstract class QueryExpressionParser : object
Remarks

Uses a tokenizer to populate a stack of tokens, which is then manipulated from the various parsing routines for subexpressions. Implementations should throw QueryParseException on invalid input.
