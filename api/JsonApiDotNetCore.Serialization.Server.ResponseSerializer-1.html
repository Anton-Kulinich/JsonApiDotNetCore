TResource

Type of the resource associated with the scope of the request for which this serializer is used.
Remarks

Because in JsonApiDotNetCore every json:api request is associated with exactly one resource (the request resource, see GetRequestResource()), the serializer can leverage this information using generics. See ResponseSerializerFactory for how this is instantiated.
