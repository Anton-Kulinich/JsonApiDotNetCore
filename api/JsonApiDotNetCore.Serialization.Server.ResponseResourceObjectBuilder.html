GetRelationshipData(RelationshipAttribute, IIdentifiable)

Builds the values of the relationships object on a resource object. The server serializer only populates the "data" member when the relationship is included, and adds links unless these are turned off. This means that if a relationship is not included and links are turned off, the entry would be completely empty, ie { }, which is not conform json:api spec. In that case we return null which will omit the entry from the output.
