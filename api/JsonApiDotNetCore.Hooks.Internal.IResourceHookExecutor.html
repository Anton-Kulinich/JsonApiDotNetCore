Interface IResourceHookExecutor

Transient service responsible for executing Resource Hooks as defined in ResourceHooksDefinition<TResource>. See methods in IReadHookExecutor, IUpdateHookExecutor and IOnReturnHookExecutor for more information. Uses JsonApiDotNetCore.Hooks.Internal.Traversal.NodeNavigator for traversal of nested resource data structures. Uses JsonApiDotNetCore.Hooks.Internal.Execution.HookContainerProvider for retrieving meta data about hooks, fetching database values and performing other recurring internal operations.
